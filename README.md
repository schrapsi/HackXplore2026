# IMPACT

## Vision
IMPACT is a modern, minimalistic web application designed for young philanthropists. Inspired by the clean aesthetics of neobrokers like TradeRepublic and ScalableCapital, IMPACT connects young, wealthy individuals with charity projects that align with their values and budgets. 

## Design Philosophy
- **Minimalistic & Clean:** A clutter-free interface focusing on the user's philanthropic goals.
- **AI-Style Interaction:** Simple, prompt-based discovery.
- **High Transparency:** Clear metrics and live updates on funded projects.

## User Journey
The platform facilitates a seamless four-step journey:

1. **Discovery (Landing Page)**
   - A minimalist search interface (similar to Google or ChatGPT) asking a single question: *"How do you want to contribute to society?"*
   - Users provide a text-based answer and select a budget tier (e.g., 20k-50k, 50k-100k, 100k-200k, 200k+).

2. **Project Matching**
   - The platform curates and presents charity projects fitting the user's prompt and budget.
   - Clean visualizations for exploration, featuring interactive modes like a map view or a swiping mechanism for curation.

3. **Commitment & Onboarding**
   - Once a project is selected, the user enters a streamlined account creation workflow.
   - The workflow culminates in a secure commitment and payment process to fund the chosen project.

4. **Dashboard & Transparency**
   - **Live Updates:** Logged-in users access a dashboard providing total transparency on how their funds are utilized.
   - **Tangible Metrics:** Continuous updates and real-time metrics on the funded projects' impact.
   - **Repeat Funding:** Users can initiate new funding processes directly from their dashboard.
